**AI-Powered Email & QR Threat Intelligence Platform**

> SENTRIX is a full-stack cybersecurity platform built to detect, analyze, and score phishing emails, malicious URLs, QR-based attack vectors, suspicious attachments, and domains — powered by real-world threat intelligence APIs and an automated multi-signal scoring engine. Built as a production-grade security tool.

**System Architecture:**

```
┌──────────────────────────────────────────────────────────────────────────────┐
│                           SENTRIX THREAT PIPELINE                            │
│                                                                              │
│   INPUT LAYER           ANALYSIS ENGINE              INTELLIGENCE OUTPUT     │
│   ────────────          ───────────────              ─────────────────────   │
│   📧 Email Headers   →  SPF / DKIM / DMARC       →  Auth Verdict            │
│   🔗 URLs            →  VirusTotal + URLScan      →  Threat Classification  │
│   📱 QR Codes        →  Decode + Pattern Match   →  Phishing Verdict        │
│   📎 Attachments     →  Hash + File Scan         →  IOC Extraction          │
│   🌐 Domains         →  AbuseIPDB + DNS Intel    →  Abuse Confidence Score  │
│                                                   →  Interactive Dashboard  │
└──────────────────────────────────────────────────────────────────────────────┘
```

| 🔗 URL Intelligence | VirusTotal · URLScan | Deep malware & phishing URL scanning |
| 📱 QR Threat Detection | QR Decoder + Pattern ML | QR code decoding + attack pattern recognition |
| 🌐 Domain Analysis | AbuseIPDB · DNS Intelligence | Reputation scoring + abuse confidence |
| 🔎 IOC Extraction | Custom Parser Engine | IP · hash · domain extraction pipeline |
| 📊 Threat Scoring | Custom Scoring Engine | Dynamic LOW / MEDIUM / HIGH risk classification |
| 🖥️ Dashboard | React + FastAPI | Real-time interactive threat visualization |
