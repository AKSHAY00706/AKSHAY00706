### 🛡 NullTrace — IP Threat Intelligence Platform

> A **professional-grade cyber intelligence platform** that analyzes suspicious IPs in real-time using live threat APIs. Built end-to-end — from Python backend to polished frontend — and deployed to production.

| Feature | Description |
|---------|-------------|
| 🔍 IP Reputation Lookup | Instant threat level assessment for any IP |
| 🧬 VirusTotal Integration | Deep malware & URL scan via VirusTotal API |
| 🚨 AbuseIPDB Risk Analysis | Real-time abuse confidence scoring |
| 📊 Bulk Excel IP Scanning | Upload & scan hundreds of IPs at once |
| 📥 Smart Report Downloads | Auto-generated Excel threat reports |
| ⚡ Live Threat Scoring | Dynamic risk levels: LOW / MEDIUM / HIGH |
| 💎 Premium Responsive UI | Polished dark-theme interface, mobile-ready |
| 🌐 Production Deployment | Hosted live on Render |
